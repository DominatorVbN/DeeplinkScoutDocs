DeeplinkScout

Test and manage deeplinks for iOS and macOS development

Download Latest

Install via Homebrew

brew tap DominatorVbN/DominatorVbN
brew install deeplinkscout

Features

  • Test deeplinks on simulators and physical devices
  • Organize deeplinks into collections
  • Export and share deeplink collections
  • Automatic updates via Sparkle

Requirements

  • macOS 14.6 (Sonoma) or later
  • Xcode command line tools
