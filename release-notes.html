DeeplinkScout v1.1.0

Version v1.1.0 • Mar 22, 2026

What's New in v1.1.0

Features
  • Automatic updates via Sparkle
  • Notarized for macOS security
  • Available via Homebrew
Installation

**Direct Download:**

  1. Download `DeeplinkScout.dmg`
  2. Open the DMG and drag DeeplinkScout to Applications
  3. Launch from Applications folder

**Homebrew:**

```bash

Tap the repository

brew tap DominatorVbN/DeeplinkScout

Install DeeplinkScout

brew install deeplinkscout

```

Or in a single command:

```bash

brew install DominatorVbN/deeplinkscout/deeplinkscout

```

Requirements
  • macOS 14.6 (Sonoma) or later
  • Xcode command line tools

---

**Full Changelog**: https://github.com/DominatorVbN/DeeplinkScoutApp/compare/previous-tag...v1.1.0
