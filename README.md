# DeeplinkScout Documentation

Documentation and downloads for DeeplinkScout.

## Installation

### Homebrew

```bash
brew tap DominatorVbN/DeeplinkScout
brew install DeeplinkScout
```

### Direct Download

Download the latest DMG from [Releases](https://github.com/DominatorVbN/DeeplinkScoutApp/releases/latest).

## Features

- Test deeplinks on simulators and physical devices
- Organize deeplinks into collections
- Export and share deeplink collections
- Automatic updates via Sparkle

## Requirements

- macOS 14.6 (Sonoma) or later
- Xcode command line tools
